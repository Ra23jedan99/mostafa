User Launches App: The user opens the Muslim Soul app on their mobile device.

Check Location Services: The app checks if location services are enabled on the user's device.

Yes: Proceed to step 3.

No: The app prompts the user to enable location services. User interaction is needed here (e.g., tapping a button to allow location access).

Access User Location: If enabled, the app retrieves the user's current location (city and country).

Prayer Time Calculation: The app uses the retrieved location and a built-in prayer time calculation algorithm to determine the prayer times for the user's specific location.

Prayer Time Display: The app displays the calculated prayer times on the app's interface, typically for Fajr, Dhuhr, Asr, Maghrib, and Isha prayers.

Optional: Settings & Customization: The user might have options to:

  • View prayer times for different calculation methods (if available).
  • Set notifications for upcoming prayer times.
